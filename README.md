# API Gateway

Backend-focused API gateway project.

The implementation currently lives in [`backend/`](./backend), including:
- application code
- config and validation
- HTTP transport and middleware
- tests
- Dockerfile and Makefile

## Quick Start

```bash
cd backend
make run
```

For full backend documentation, see [`backend/README.md`](./backend/README.md).

## CI/CD
